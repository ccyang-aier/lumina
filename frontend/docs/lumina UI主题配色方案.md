# Lumina UI 主题配色方案

写在前面：lumina的前端ui配色方案崇尚黑/白经典简约色系，彩色的使用应当保持克制，可在关键ui元素等设计处选择用彩色点缀以增加视觉效果。

## 1. 主色系：Sky（天空蓝）

- **清新且灵动**：Sky 在浅色阶下呈现出通透、明亮的质感，符合 Lumina “知识、光、连接”的意象，比 Violet 更具科技感和未来感。
- **统一且不混杂**：全站彩色 UI（按钮、链接、高亮、边框等）统一使用 Sky 色相，避免混用 Blue/Indigo/Cyan 等相近色。
- **浅色系策略**：重点使用 50-400 的浅色阶作为背景、边框和装饰，营造清新氛围；交互元素使用深色阶或高对比度色以保证可读性。

## 3. Light / Dark 两套配色建议

### Light

- **背景**：纯白（#ffffff）或 primary-50（极淡蓝）。
- **文本**：深灰/黑（foreground）；避免使用纯黑，推荐 slate-900 或 neutral-900。
- **主色应用**：
    - **按钮**：背景使用 primary-500，文字使用白色（确保对比度）。
    - **高亮/选中**：使用 primary-100 作为底色，primary-300 作为边框。
    - **链接**：primary-600（深蓝）以确保可读性。

### Dark

- **背景**：深灰/黑（background = 9% Lightness）。
- **文本**：纯白/浅灰（foreground）。
- **主色应用**：
    - **Logo**：必须为 **白色** 或 **浅灰渐变**，确保在深底上清晰可见。
    - **按钮/高亮**：使用 primary-500（天空蓝）作为主色，利用发光效果（glow/shadow）增加层次。
    - **边框**：primary-500/30 (30% 透明度) 或 gray-800。

## 4. “高级感场景”清单与判别规则

### Logo方案

- **Light 模式**：
    - **颜色**：深灰至黑色渐变（`from-gray-900 via-gray-800 to-gray-600`）。
    - **特效**：使用 `bg-clip-text` 实现金属质感渐变；Hover 时添加细微缩放（scale-105）和底部光标动画。
- **Dark 模式**：
    - **颜色**：**白色至浅灰渐变**（`from-white via-gray-100 to-gray-300`）。**绝对禁止使用黑色**。
    - **特效**：文字本身发光（drop-shadow），保持高对比度。

### 其它“高级感场景”

- **适用范围**：Navbar 品牌区、Hero Section 标题、Slogan、重要数据展示。
- **设计手法**：
    - **黑白灰主调**：使用无彩色的黑白灰渐变来传达专业、稳重。
    - **动效点缀**：在 Hover 或交互时，引入 Sky 主色作为“点睛之笔”（例如下划线、光晕、边框高亮）。
    - **字体排印**：增加字间距（tracking-tight/wide），使用粗体（Bold/Extrabold）配合阴影。

## 5. 注意事项

1.  **对比度检查**：在 Light 模式下使用 primary-300/400 作为文本时需格外小心对比度，建议仅用于大字号标题，正文请用 primary-600+ 或灰色。
2.  **Dark 模式适配**：务必检查 Logo 和黑色文字在 Dark 模式下是否反白。使用 `dark:text-white` 或 `dark:from-white`。
3.  **统一色相**：清理代码中残留的 violet/purple 等颜色，统一替换为 sky 系列。
